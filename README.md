# Website Data Downloader

PowerShell скрипт для полного скачивания веб-сайтов со всеми ресурсами

## Возможности

- Рекурсивное скачивание HTML страниц
- Загрузка CSS, JavaScript, изображений и других медиафайлов
- Сохранение оригинальной структуры папок сайта
- Ограничение глубины обхода ссылок
- Информация о статусе и прогрессе загрузки
- Просмотр метаданных страницы без скачивания

## Поддерживаемые типы файлов

| Тип | Расширения |
|-----|-------------|
| Веб страницы | .html, .htm, .xml |
| Стили | .css |
| Скрипты | .js |
| Данные | .json, .xml |
| Изображения | .jpg, .jpeg, .png, .gif, .svg, .webp, .ico |
| Шрифты | .woff, .woff2, .ttf |
| Документы | .pdf |

## Системные требования

- Windows PowerShell 5.1 или выше
- .NET Framework 4.5 или выше

## Установка

Скачайте файл `download.ps1` в любую папку

## Использование

### Запуск скрипта

```powershell
.\download.ps1
